# capstone2
Capstone 2 Project, Interpretable Machine Learning Models for Healthcare Outcome Prediction Coding Process

THE FILES IN THIS PROJECT ARE NOT SUPPLIED! THEY ARE FROM THE MIMIC-III REPOSITORY, WHICH YOU NEED SPECIAL ACCESS TO OBTAIN. SEE MORE HERE: https://physionet.org/content/mimiciii/1.4/
Libraries required (Also stored in 00_libraries.py):

Core libraries:
    pip install --upgrade pip 
duckdb, polars, pyarrow, pandas, pyreadstat, joblib, openai

Modeling (for later phases): scikit-learn, lightgbm, xgboost, shap, matplotlib, seaborn

You also need a valid OpenAI key to have the LLM functions to work!
Directories are omitted. They must be changed according to your own needs.
Feature Dictionary included, but directory must also be changed.
